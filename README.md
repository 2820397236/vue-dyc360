# 使用技术

> vue2 + vuex + vue-router + webpack + ES6/7 + fetch + sass + flex + svg

## 说明
# 该应用公供学习参考

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
